# Actividad Obligatoria 3 - Single Page Application (SPA)

## Descripción

Aplicación desarrollada con React y TypeScript que permite consultar información de personajes mediante una API pública.

### Funcionalidades

* Navegación SPA mediante React Router.
* Búsqueda de personajes.
* Visualización de detalle.
* Sistema de favoritos utilizando Context API.
* Custom Hook para peticiones HTTP.
* Página 404 personalizada.
* Login simulado con validaciones utilizando React Hook Form.

## Tecnologías utilizadas

* React
* TypeScript
* Vite
* React Router DOM
* React Hook Form

## API utilizada

**Rick and Morty API**

URL Base:

https://rickandmortyapi.com/api

La API utilizada es pública y no requiere API Key.

## Instalación

Clonar el repositorio:

```
git clone URL_DEL_REPOSITORIO
```

Instalar dependencias:

```
npm install
```

Ejecutar la aplicación:

```
npm run dev
```

Generar versión de producción:

```
npm run build
```
